# ムームードメイン固有手順

**方式**：Gmail 共通操作 + レシピ B（メール本文を PDF 化）

ムームードメインは**請求書 PDF の発行を行っていない**。送信される**メールがインボイス（適格請求書）として利用可能**。領収書相当も同メールに含まれるため別途生成しない。

## 対象メール

送信元 `no-reply@service.muumuu-domain.com`：

| 件名パターン | 種類 | 頻度 |
|------------|------|------|
| `【ムームードメイン】ご契約完了のお知らせ` | 新規ドメイン契約時 | 契約時1回 |
| `【重要】【ムームードメイン】自動で入金・更新が完了しました` | 自動更新時 | 年1回 |

本文内の **「■更新ドメイン」「【 ドメイン 】」** 行で対象ドメインを判定する。

## 手順

1. Gmail 検索：`from:muumuu-domain.com`
2. 対象年の「自動更新完了」または「ご契約完了」メールを抽出し、各メールを開いて本文からドメインを確認。
   処理対象は `local.yml` の `services.moomoo.domains` に登録されたドメインのみ（登録外は除外）

```javascript
// 一覧行の特定例（件名キーワード＋日付でフィルタ）
const target = rows.find(r => /自動で入金・更新が完了/.test(r.textContent) && /<対象の日付>/.test(r.textContent));
// 本文からのドメイン抽出例
const domainMatches = body.textContent.match(/[a-z0-9-]+\.(com|net|org|jp)/gi);
```

3. **`window.print()` を抑制してから**「すべて印刷」をクリック（Gmail 共通操作参照）→ 印刷タブでレシピ B（`format: 'A4'`、margin 15mm）
4. リネーム移動 → 印刷タブを閉じて次のメールへ

## ファイル名の `{domain_fqdn_us}`

- 処理対象ドメインの `fqdn` の `.` を `_` に置換した値（例：`example.com` → `example_com`）。
  ドメイン名自体のハイフンはそのまま（`my-example.com` → `my-example_com`）

## 注意・落とし穴

- メール本文には他の個人情報が含まれる可能性があるので、本文の範囲を確認する
- PDF は複数ページになることがある
