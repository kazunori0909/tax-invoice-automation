# マネーフォワード クラウド（サブスク利用料）固有手順

**方式**：レシピ B（請求書は HTML のみ。直接ダウンロードボタンは無い）

## 手順

1. `https://erp.moneyforward.com/office_usage_detail_statements` を開く
   - 未ログイン時は MF ID ログインに遷移：「アカウントを選択する」→ 現在のアカウントを選択。
     パスワード入力が出たら**ユーザーにログインを依頼して停止**
   - ログイン後 `/loginable_office_selection/edit` で事業者一覧が出たら、会計 MCP と同じ事業者
     （`local.yml` で利用中の事業者）の「選択する」をクリック
   - 「事業者◯◯にログインしてもよろしいですか？」の confirm は accept（`browser_handle_dialog accept=true`）
   - ホームに入ったら「料金明細」タブへ
2. **対象行の特定**：各行は `請求日 / ご利用料金 / 内訳・請求書`。
   年額契約は金額が年額（月額より大きい）の行が年 1 件、月額契約は毎月末の行
3. 「請求書」リンクをクリック → 新規タブ（index 1）で請求書 HTML が開く
4. レシピ B で PDF 化。**請求書本文は幅 1200px** のため A4 既定だと右端（請求書ID・登録番号）が
   見切れる。以下のレイアウトを指定する：

```
width: '1240px', height: '820px', printBackground: true,
margin: { top: '12px', bottom: '12px', left: '20px', right: '20px' }
```

5. リネーム移動 → 印刷タブを閉じる

## 年月の導出（重要）

ファイル名の年月は**請求書の発行日基準**（請求日ではない）。
年額は「請求日 = 前年末／発行日 = 翌年初」のように年をまたぐことがあり、仕訳も発行日（取引日）で計上する。
例：請求日 2025-12-31／発行日 2026-01-05 → `2026-moneyforward-invoice.pdf`（FY2026）。

## 確認済み実例（初回実行・年額）

- 請求書 URL：`/office_usage_detail_statements/<hash>/bills`（HTML）
- 料金明細に年額・月額が混在（年額切替後は年1件の大きい金額、切替前は毎月末の月額）
- 契約期間は請求書 PDF には記載されず、`erp.moneyforward.com/home` の「契約内容」に表示される
