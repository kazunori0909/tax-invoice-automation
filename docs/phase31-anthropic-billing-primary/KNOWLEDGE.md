# Phase 31 KNOWLEDGE

## claude.ai Billing / Stripe 請求書の日付表示

- Stripe ホスト型請求書ページ（日本語 UI）に "Date of issue" という項目は存在しない。表示されるのは「支払い日」のみ
- claude.ai の請求書テーブルの「日付」列と、Stripe 請求書側の「支払い日」は**最大1日ズレる**ことがある
  （2026-07-08 実機確認：テーブル表示「2026年6月1日」、Stripe「支払い日 2026年5月31日」）。
  契約更新日が月境界付近のため、タイムゾーン差で前日表示になったと推定される
- MF 連携のカード明細の取引日と一致するのは **claude.ai テーブル側の日付**。年月導出はこちらを基準にする
